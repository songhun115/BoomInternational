BIT Vision & Background

BIT 토큰은 현재의 메타버스 시장에 존재하는 문제를 해결하기 위한 솔루션을 제공하는 프로젝트로써, 독자적인 개발을 통해 메타버스 플랫폼 환경과 온라인 서비스에서 상호 교환이 가능한 형태의 블록체인 네트워크로 구현될 것이다.

Development Roadmap

Stage I : 2022 4Q

  • BIT 토큰 가상화폐 거래소 상장
  • BIT 플라즈마 모듈 프로토타입 출시, 아토믹 스왑 테스트 진행
  • 스마트폰 기반 환경에서 3D 데이터 맵핑을 위한 Apple ARKit 6 R&D

Stage Ⅲ : 2023 4Q

  • BIT IDC센터 클라우드 컴퓨팅 인프라 구축
  • BIT 서버리스 Web RTC 모듈 출시
  • BIT 솔루션 고객사에게 모듈 호스팅을 통한 서비스 제공 시작

Stage Ⅴ : 2024 4Qh

  • 풀 다이브 기반 BIT 메타버스 플랫폼 출시

Stage Ⅱ : 2023 2Q

  • BIT 플라즈마 모듈 데이터 익스플로러 출시
  • BIT 플라즈마 모듈 정식 버전 출시
  • Point Cloud & LiDAR 센서 기반 3D 아바타 생성 모듈 출시

Stage Ⅳ : 2024 3Q

  • BaaS 서비스 확장을 위한 메인넷 업데이트 진행
  • 메타버스 플랫폼 내 사용자 아바타에 DID 적용을 위한 신원증명 모듈 프로토타입 출시
  • XR, VR 및 풀 다이브 하드웨어 기반 소프트웨어와 연동을 위한 선행 연구 진행
